# TinyBookmarker

TinyBookmarker is a tiny self-hosted bookmark manager for checking, cleaning and managing your bookmarks.

**Tagline:** Check, clean and manage your bookmarks.

## What TinyBookmarker is

TinyBookmarker is built for people who want to:

- save bookmarks quickly
- keep favorites visible and useful
- organize bookmarks with collections and tags
- search, filter and sort efficiently
- review and clean up duplicates safely
- import and export bookmark data without losing trust
- keep the whole product lightweight and practical for self-hosting

## What TinyBookmarker is not

TinyBookmarker is **not primarily**:

- a read-later app
- a reader-first product
- an archive-first product
- a knowledge-base-first tool
- a note-taking-first tool
- a technical dashboard

Archive, reader or AI-heavy features may be explored later, but they are not the product identity.

## Core product areas

TinyBookmarker is centered around these main areas:

1. Bookmarks
2. Favorites
3. Collections
4. Tags
5. Duplicates
6. Import / Export

## Core workflows

### Save
- quick add
- sensible defaults
- duplicate preflight
- inbox / triage support

### Organize
- favorites
- collections
- tags
- search / filter / sorting

### Clean
- duplicate review
- safe merge
- undo-friendly flows
- bulk cleanup support

### Migrate
- import
- export
- browser roundtrip support
- provenance-aware workflows

## Early scope

Version 1 should focus on:

- bookmark saving and editing
- favorites
- collections and tags
- duplicate handling
- import / export
- browser roundtrip basics
- lightweight self-host deployment
- clear, calm UI / navigation

## Self-hosting direction

TinyBookmarker should stay:

- lightweight
- easy to update
- easy to back up
- practical for small self-hosted setups
- realistic for Proxmox / Debian LXC

## Guiding rule

Before adding a feature, ask:

**Does this make TinyBookmarker better at checking, cleaning and managing bookmarks?**

If not, it is probably too early or the wrong direction.
